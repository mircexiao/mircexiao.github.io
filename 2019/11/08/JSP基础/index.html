jsp基础

  1. 变量和方法声明：<%! int i=6;%>

  2. 表达式：<%=表达式%>

  3. JSP指令

    • page指令
    • include指令：<%@ include file=”相对路径”%>
    • taglib指令：<%@ taglib uri=”URIToLibrary” prefix=”标记前缀”%>

  4. JSP动作

    • jsp:include:在页面上动态的插入文件

    • jsp:forward:转向另一个页面

    • jsp:plugin:将服务端的Java小应用或JavaBean组件下载到浏览器去执行

    • jsp:useBean:用来装载一个将要在JSP页面使用的JavaBean

      有效范围取值：page、request、session、application

    • jsp:setProperty:用来设置已经实例化的bean对象的属性

      或用request参数值设置JavaBean属性值

内置对象

内置对象作用范围

  1. application：从服务器开始到服务器关闭

  2. session：从客户端与服务器连接开始到连接中断为止

  3. Request：从一个jsp向另一个jsp请求开始，之后中断

  4. Page：作用范围为当前页

out对象
