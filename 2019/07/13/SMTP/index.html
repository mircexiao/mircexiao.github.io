#Python 发送邮件

python可使用SMTP协议发送邮件，python内置smtplib模块，无需下载。

主要功能代码示例：


    import smtplib
    server = smtplib.SMTP(smtp,25)
    server.login(sendadress,password)
    server.sendmail(sendadress,toadress,msg.as_string())
    server.quit()

简单测试：


    import smtplib
    from email.mime.text import MIMEText

    mailserver = "smtp.163.com"  #邮箱服务器地址
    username_send = '15606161139@163.com'  #邮箱用户名
    password = 'wulian1111'   #邮箱密码：需要使用授权码
    username_recv = '3206403613@qq.com'  #收件人，多个收件人用逗号隔开
    mail = MIMEText('这是我发的邮件')
    mail['Subject'] = '邮件主题'
    mail['From'] = username_send  #发件人
    mail['To'] = username_recv  #收件人；[]里的三个是固定写法，别问为什么，我只是代码的搬运工
    smtp = smtplib.SMTP(mailserver,port=25) # 连接邮箱服务器，smtp的端口号是25
    # smtp=smtplib.SMTP_SSL('smtp.qq.com',port=465) #QQ邮箱的服务器和端口号
    smtp.login(username_send,password)  #登录邮箱
    smtp.sendmail(username_send,username_recv,mail.as_string())# 参数分别是发送者，接收者，第三个是把上面的发送邮件的内容变成字符串
    smtp.quit() # 发送完import smtplib
    print ('success')

邮箱密码需使用邮箱授权码，需要自己在相应的邮箱里设置开启。

MIMEText()中三个参数为邮件发送内容，文本格式(plain/html)以及编码格式。
