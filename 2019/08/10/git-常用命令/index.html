git 常用命令

Aug 10, 2019Hits

简介

git为分布式版本控制工具，工作流程为先将工作区文件放入暂存区，最后在提交到版本库中。

基本命令

创建一个仓库

git init

把文件添加到版本库中

git add

提交文件

git commit -m<提交说明>

参数-m后为每次提交后的说明

查看提交状态

git status

查看提交日志

git log

修改文件名

git mv

参数

-p 查看变动信息

-p -1 查看最近一次提交

–oneline 显示简短信息

–name-only 显示发生变动的文件

–name-status 显示修改类型

修改最近一次提交

git -commit –amend

分支处理

创建分支

git branch

切换分支

git checkout

以上两条命令可以简写为

git checkout -b

即创建并切换到该分支

删除分支

git branch -d

合并分支

将分支合并到主分支中去

git merge

查看已合并的分支

git branch –merged

查看未合并的分支

git branch –no-merged
